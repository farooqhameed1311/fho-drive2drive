User Guide

Learn how to effectively use the transfer engine.

Cloud to Cloud Transfers

1 Get the Source Link

Go to your Google Drive, right-click the file or folder you want to transfer, click "Share", and ensure General Access is set to "Anyone with the link". Copy that link.

2 Set the Destination (Optional)

If you want the file saved in a specific folder, get the share link for that target folder (ensure it is set to Editor). If you leave this blank, the file will be saved securely to our root server and a direct download link will be provided to you instantly.

Device Uploads

1 Select Your File

Toggle to the "Device Upload" tab in the engine. Click the upload zone to browse your computer or mobile device, or simply drag and drop the file into the box.

2 Wait for Encryption

Our server will encrypt and upload the file using a secure Base64 stream. Please do not close the window while the transfer is processing. Once complete, you will receive a secure URL to access your file.
